## FAQ

### CN2 GIA 和 CN2 GT 有什么区别，买 VPS 时怎么判断？

CN2 GIA 是全程 GIA（Global Internet Access）线路，去程和回程都走 CN2 骨干网，IP 段通常是 59.43.x.x。CN2 GT 只有部分路径走 CN2，高峰期容易绕道普通公网。

判断方法：买之前看商家是否明确标注"CN2 GIA 双向"，买之后用 traceroute 看回程路由，出现 59.43 开头的节点就是走了 CN2 GIA。DMIT 的 Pro 系列明确标注 CN2 GIA，EB 系列是 CMIN2 为主的混合线路。
